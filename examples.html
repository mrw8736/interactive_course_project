Examples

Let's go through a few examples to see what each selector looks like and how they differ from one another.

Example 1 - Type Selector
This one is a type selector because it doesn't have a pound or period preceding its name, and it contains the name 'div', meaning this will target every 'div' element and apply the color style of red to it.
Example 2 - Class Selector
At first glance you may think this is the same as the last example, but don't let names fool you! Even though it's using the name of an HTML element, the selector is still a class selector because the 'div' is preceded by a period. Therefore, it will target every element that has the class attribute of 'div' instead of all 'div' elements.
Example 3 - ID Selector
This one is an ID selector because it has a pound symbol preceding its name, and it contains the name 'bell'. Because we are dealing with IDs, this means this will only target the one element with the ID 'bell'.
Example 4 - Grouped Selector
This one has the ID selector for the 'bell' ID from the last example, alongside a class selector for the 'taco' class. Because they're separated by a comma and next to the same style, they are grouped, meaning the element with the 'bell' ID and all elements of the 'taco' class will get the color style of red.
