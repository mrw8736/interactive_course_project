Unit 2.2: Style Selectors

Objective Overview Instruction Examples Practice Test

Requirements/Entering Skills

Before beginning this module, you should be able to link CSS files in your directory to HTML files by nesting a link tag in the head tag of an HTML file as per what was covered in Unit 2.1. If you do not have this skill or have not completed the unit, please return to the previous unit to fully complete it before you continue.

Objective

Given a list of 8 CSS HTML selectors, you will be able to identify what element or set of elements a style is going to affect. You must answer 75% of the list items correctly to pass the module.
