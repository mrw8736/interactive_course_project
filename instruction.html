Instruction

Selectors are used to in CSS to find the HTML elements you want to apply your styles to. They are “selecting” the element you want and adds the styles to it.

There are three types of selectors you will be using, type selectors, class selectors, and id selectors.

Type Selectors

Type selectors are the most basic selector. They use the name of an HTML element tag and apply the style to every element with that tag.

Syntax: html_tag {style: value;}

Class Selectors

Classes are attributes that can be added to any HTML element to categorize it. You can make the class name whatever you want and you can have as many HTML elements with same class as you'd like. It is added by adding the attribute to the element's tag like so:

< tag class = 'class_name' >

To target elements with these classes, you will use class selectors, selectors that target all elements with the same class attribute and apply the style to those elements. The difference between the syntax of a class selector and a type selector is that the class name must be preceded by a period (.).

Syntax: .class_name {style: value;}

ID Selectors

IDs work similarly to classes, as they are attribute that can be added to HTML elements and the ID name can be anything you want. Unlike classes, unique IDs can only be applied to one element at a time. They should only be used for elements you plan to single out in your HTML. You can add the attribute to the element's tag like so:

< tag id = 'id_name' >

To target an element with an ID, you will use id selectors, selectors that target the element with the same ID attribute and apply the style to that element. The difference between the syntax of a id selector and other selectors is that the ID name must be preceded by a pound symbol (#).

Syntax: .id_name {style: value;}

Grouping Selectors

One last thing to know about is grouping selectors. If you want multiple element types, classes, and/or IDs to have the same styles, you can save time and space by applying the styles to a group of selectors. Simply list out the selectors and divide them with commas, like so:

Syntax: selector1, .selector2, #selector3 {style: value;}
